# Onion Weed Classification using CNN

This project uses Convolutional Neural Networks (CNN) to classify images of onion crops and weeds. The system helps farmers automatically detect weeds in onion fields.

## Features
- Image classification using CNN
- Detect onion plants vs weeds
- Data preprocessing and augmentation
- Model training and evaluation
- Prediction system

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## Dataset
Images of onion plants and weeds are used to train the CNN model.

## Model Architecture
Convolutional Neural Network (CNN) with:
- Convolution layers
- MaxPooling layers
- Dense layers
- Softmax classification

## Results
The trained model achieves high accuracy in detecting onion crops and weeds.
